Hi! Welcome to xdm's web Page. xdm is a modern MDX compiler. I was made in February '21. xdm is an MDX compiler that focusses on two things: a) Compiling the MDX syntax (markdown + JSX) to JavaScript, b) Making it easier to use the MDX syntax in different places. It boasts a couple of cool features, such as source maps, ESM only, defaulting to an automatic JSX runtime, no Babel, smallish browser size, more docs, import/exports in evaluate, esbuild and Rollup plugins.
To learn more about me, check out my GitHub page.
